Sobre

Sistema para ajudar a gerenciar despesas, organizar e controlar gastos e receitas


Este sistema faz parte de um trabalho para curso de pós graduação, usando framework Angular para desenvolvimento das atividades.
Todos os dados aqui informados, são fictícios.

Angular é uma plataforma de aplicações web de código-fonte aberto e front-end baseado em TypeScript liderado pela Equipe Angular do Google e por uma comunidade de indivíduos e corporações.
